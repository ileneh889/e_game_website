# 「龍與地下城」遊戲官方網站
## 🏆 創作理念  
線上遊戲官方網站，透過官方直播遊戲實況、定期舉辦玩家作品投票及即時更新遊戲最新資訊等核心功能，凝聚玩家向心力及提升社群參與感。


<br><br>

## 🔥 特色功能  
- 🌟 功能 1：玩家二創作品競賽與投票   [負責區塊!]
- 🚀 功能 2：遊戲實況與即時聊天
- 📜 功能 3：商店街
- ⚡ 功能 4：最新資訊公布與更新
- ⚙️ 功能 5：會員專區

### 1️⃣ 啟動專案 
*前端部分*
- 首先，使用以下指令clone專案：
```
git clone https://github.com/ileneh889/e_game_website.git
```
- 開啟專案進到terminal，安裝dependencies：
```
npm install
```
- 啟動前端服務：
```
npm run dev
```

*後端部分*
- 開啟專案進到terminal，安裝dependencies
```
npm install
```

- 資料庫環境設置： ./src/secret/secret.ts
- 啟動後端服務
```
npm run dev
```
